Uchiwa is a simple dashboard for Sensu, built with Go and AngularJS.

Essentially, Uchiwa takes the results returned by Sensu API and display these in a pleasing way to the eye so you can use Sensu to its full potential.

Uchiwa is composed of a Go web server, which query every Sensu API and expose the results through its own API to the Angular web dashboard.
