Features

Uchiwa is packed with awesome features. Check out some of the latest!


Silencing

Uchiwa provides an easy mechanism to silence any check or client for a determined or unlimited period of time. Context can be easily added by supplying a reason upon stash creation.

Filtering

Uchiwa makes it easy to filter all Sensu data based on any properties, such as a team name or a client subscription. The filtered results can also be shared with a simple link.

Unified Client

Uchiwa aggregates all information concerning a specific client within an unique view. Custom content can easily be added to a client or an event through Sensu in order to provide a better context.

Rich Data

Display rich data directly into Uchiwa, including HTML links and images, by adding an URL into a custom key-value of a check definition. See the Sensu documentation for more information.
